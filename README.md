## ✨ Features

- 🔍 **Search Albums**: Browse and search for any album using the Spotify Web API
- 📚 **Personal Library**: Keep track of your physical CD collection with album artwork
- 📝 **Wishlist**: Create a wishlist for albums you want to buy
- 📱 **Cross-platform**: Works on iOS, Android, macOS, Windows, and Web
- 🔐 **Secure Authentication**: Google Sign-In integration via Firebase
- ☁️ **Cloud Sync**: Your data is synced across all your devices
- 🎯 **Drag & Drop**: Reorder your collection as you like

## 🚀 Getting Started

### Prerequisites

- **Flutter SDK**: [Install Flutter](https://flutter.dev/docs/get-started/install) (version 3.8.1 or higher)
- **Firebase Account**: Create a project at [Firebase Console](https://console.firebase.google.com)
- **Spotify Developer Account**: Register at [Spotify Developer Dashboard](https://developer.spotify.com)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/lauraharoescoi/cd_tracker.git
   cd cd_tracker
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Firebase Setup**
   
   The project is already configured with Firebase, but you'll need to connect it to your own Firebase project:
   
   - Install Firebase CLI: `npm install -g firebase-tools`
   - Run: `firebase login`
   - Run: `flutterfire configure`
   - Select your Firebase project or create a new one
   - Enable **Authentication** and **Firestore** in your Firebase console
   - Enable **Google Sign-In** in Firebase Authentication settings

4. **Spotify API Setup** 🎵
   
   Create your Spotify API credentials:
   
   - Go to [Spotify Developer Dashboard](https://developer.spotify.com/dashboard)
   - Create a new app
   - Copy your **Client ID** and **Client Secret**
   - Create the credentials file:
   
   ```bash
   # Create the config directory
   mkdir -p lib/config
   
   # Create the credentials file
   touch lib/config/spotify_credentials.dart
   ```
   
   Add your credentials to `lib/config/spotify_credentials.dart`:
   ```dart
   // lib/config/spotify_credentials.dart
   
   // Your Spotify API credentials
   // Get these from: https://developer.spotify.com/dashboard
   
   const String spotifyClientId = 'YOUR_CLIENT_ID_HERE';
   const String spotifyClientSecret = 'YOUR_CLIENT_SECRET_HERE';
   
   // For Google Sign-In (get from Firebase Console -> Authentication -> Sign-in method -> Google)
   const String googleServerClientId = 'YOUR_GOOGLE_SERVER_CLIENT_ID_HERE';
   ```

5. **Run the app**
   ```bash
   flutter run
   ```

## 🔧 Configuration

### Required Files

The following files are needed but are not included in the repository for security reasons:

- `lib/config/spotify_credentials.dart` - Your Spotify API credentials
- `lib/firebase_options.dart` - Generated by FlutterFire CLI
- `android/app/google-services.json` - Firebase config for Android
- `ios/Runner/GoogleService-Info.plist` - Firebase config for iOS
- `macos/Runner/GoogleService-Info.plist` - Firebase config for macOS

### Google Sign-In Setup

To enable Google Sign-In:

1. In Firebase Console, go to **Authentication** → **Sign-in method**
2. Enable **Google** as a sign-in provider
3. Copy the **Web SDK configuration** → **Web client ID**
4. Add this as `googleServerClientId` in your credentials file

### Spotify API Setup

1. Create a Spotify app at [developer.spotify.com](https://developer.spotify.com/dashboard)
2. No redirect URIs needed (we use Client Credentials flow)
3. Copy the **Client ID** and **Client Secret**
4. Add them to your credentials file

## 🎯 Usage

1. **Sign In**: Use your Google account to sign in
2. **Search**: Tap the search icon to browse albums
3. **Add to Library**: Found a CD you own? Add it to your library
